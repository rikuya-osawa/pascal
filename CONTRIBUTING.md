# Mental Model Atlas への貢献ガイド

「日本語圏初の体系的なメンタルモデル辞書」を共に創り上げてくださる皆さんの力を必要としています。
本プロジェクトは、単なる用語集ではなく、意思決定を支える「知的OS」を目指しています。

## 🤝 貢献の種類
1. **モデルの新規追加**: 未掲載のメンタルモデルの執筆。
2. **コンテンツのブラッシュアップ**: 既存モデルの誤用例、実践例の追記。
3. **英語翻訳の対応・改善**: 機械翻訳的な表現を、より自然な英語へ修正。
4. **バグ報告・機能提案**: UI/UXの改善提案や不具合の報告。

## 📝 執筆の指針（Quality Standard）
コンサルタントの視点を軸とした「実践的な知恵」を重視します。
- **客観性**: 主観的な意見だけでなく、普遍的な定義をベースにする。
- **実用性**: 「どう使うか」だけでなく「どう間違えやすいか（誤用例）」を必ず含める。
- **簡潔さ**: 冗長な説明を避け、構造化（箇条書き等）を意識する。

## 🚀 ワークフロー
1. **Issueを確認**: 重複を避けるため、既存のIssueを確認するか、新規追加の提案をIssueに立ててください。
2. **Fork & Branch**: 本リポジトリをフォークし、`feature/add-model-name` のようなブランチで作業してください。
3. **Pull Request**: テンプレートに従ってPRを作成してください。
